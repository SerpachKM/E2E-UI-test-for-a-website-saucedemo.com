# E2E Test for saucedemo.com

## Требования:
- Python 3.7+
- Установите зависимости:
  ```bash
  pip install -r requirements.txt
  ```

## Запуск:
1. Активируйте виртуальное окружение:
   ```bash
   source venv/bin/activate  # Mac/Linux
   venv\Scripts\activate  # Windows
   ```

2. Запустите тест:
   ```bash
   pytest test_purchase.py
   ```
